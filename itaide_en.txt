# Ita IDE Plug-in for RAD Studio

## What's this?

This plug-in displays the image in the editor background.
Supported version: XE2 - 10.4 Sidney

## Installation

1. Open the package "src\itaide.dpk"
2. Right click on "itaide.bpl" and click "Install"

## Change the image

Go to "Tools -> Ita IDE Options"
Supported file formats: Bitmap, JPEG, PNG

Default image is [Pronama-Chan](http://pronama.azurewebsites.net/pronama/).
